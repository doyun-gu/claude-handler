Search the Notion workspace for "$ARGUMENTS" using Notion MCP.

1. Run a broad search with the query
2. If results are thin, try alternate keywords and synonyms
3. For each result, show:
   - Page title
   - Parent (which project/section it lives under)
   - Type (page / database entry)
   - Brief content preview (first 2 lines)

Print results as a compact list. If no results found, suggest related searches.
