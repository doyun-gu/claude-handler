# Template: Lecture Notes

Use this template for university lectures, seminars, tutorials, or any structured learning content.

---

## Notion Structure

```
# [Course Code] — [Lecture Topic]

[/callout — grey, no icon]
Course: [Course Name] | Lecturer: [Name]
Date: YYYY-MM-DD | Week [N] / Lecture [N]
[/end callout]

---

## Summary

3–5 sentences. What was covered, the main takeaway, and how it connects
to previous material. Write this as if explaining to a classmate who missed
the lecture.
```

## Database Entry

When creating a lecture note, add an entry to the `[Course] Lectures` database:

| Property | Value |
|----------|-------|
| Name | [Course Code] — [Topic] |
| Status | Done (if complete) / Draft (if incomplete) |
| Date | Lecture date |
| Week | Week number |
| Tags | Topic keywords |
| Summary | One-sentence summary |
| Gaps | Number of unresolved questions (0 = fully understood) |
